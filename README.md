# Nmap Scanner Web App

A lightweight web-based tool that automates Nmap scans to identify open ports, services, and host details.  
Built with **Python (Flask)** for backend scanning and **HTML, CSS, JavaScript** for an interactive frontend.  
Designed for learning and demonstrating network reconnaissance workflows in ethical cybersecurity environments.

## Features
- Automated Nmap scanning from a web interface  
- Displays open ports and service details  
- Simple, lightweight Flask-based backend
